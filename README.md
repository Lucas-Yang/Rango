# rango
## 项目介绍
**rango** 名字来源于电影《兰戈》， 主要是想致敬Django

平台功能集成：

- master 发布视频/图片/音频 标注任务
- 外包/众测 领取标注任务标注
- 视频/图像/音频 自动化评估(主客观 / 有无参考)
